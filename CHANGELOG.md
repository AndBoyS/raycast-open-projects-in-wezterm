# Open Projects on IDE Changelog

## [Initial Version] - 2025-03-04

### Added
- Initial release of the Raycast extension
- Project discovery functionality
  - Auto-detection of various project types:
    - Git repositories (.git)
    - Node.js projects (package.json)
    - Rust projects (Cargo.toml)
    - Go projects (go.mod)
    - Python projects (requirements.txt)
- IDE integration support
  - Visual Studio Code with ⌘ + Return shortcut
  - Windsurf with ⌥ + Return shortcut
  - Cursor with Return shortcut
- Configurable workspace path in extension settings
- Project list interface with path display
- Success/failure notifications when opening projects
